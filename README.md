Appcast XML package for Go
==========================

```
appcast -append  \
	-title="1 bug fix" \
	-description="release note" \
	-file tests/tests.zip \
	-url http://yourhost.com \
	-version 109 \
	-dsaSignature="blah" \
	-versionShortString="1.4" \
		tests/appcast.xml
```
